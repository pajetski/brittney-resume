# Brittney Chase Resume Website

Static resume and professional presence site for Brittney Chase.

## Structure

- `index.html`: Main website page.
- `assets/css/main.css`: Site styling.
- `assets/js/main.js`: Small browser enhancements.
- `assets/favicon`: Favicon, app icons, and web manifest.
- `assets/images/originals`: Preserved source images with descriptive labels.
- `assets/images/web`: Optimized JPG and WebP images for the live site.
- `assets/video/originals`: Preserved source video.
- `assets/video/web`: Compressed video and poster image for the hero.
- `assets/docs`: Resume PDF.

## Image Labels

- `brittney-chase-headshot`: Primary professional portrait.
- `brittney-chase-creative-technology-workshop`: Arts and technology planning scene.
- `encinitas-civic-arts-center`: Civic arts and coastal context image.
- `brittney-chase-ai-arts-workflow`: AI-assisted creative workflow image.
- `brittney-chase-public-art-walk`: Public art leadership image.
- `brittney-chase-community-arts-planning`: Community arts planning image.
